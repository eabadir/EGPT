18.313 — Introduction To EGPT

The New Manhattan Project

John von Neumann’s “Computer & The Brain” project to build an AI computing architecture that takes 20 watts instead of 20 Giga Watts.

“Imagine there was an AI computer which ran on the power of a dim refrigerator lightbulb but out-performed the most massive of data centers? That computer exists and it is called the human brain.

This project is a job offer, not a handout. We are building the foundation of John von Neumann’s hyper-efficient AGI computing architecture laid out in “The Computer & The Brain”. The contents of this repository show not only that it can be built but it provides working foundations to build it. You’ll find formalized machine checked proofs, computational simulation derivations of the necessary physics, and working pedagogical code libraries.

The computational approach was first ideated by Stanislaw Ulam, then formalized and existentially proved by Gian-Carlo Rota, and now constructively extended by Essam Abadir. The work is only beginning and you are invited to join.”

— README.md

Modules

Module 1: The Proof Chain

Formal Proofs (Lean 4) — Machine-verified theorems in Lean 4 with mathlib4. No sorry. No custom axioms. The constructive proof that P = NP, Rota's Entropy Theorem formalized, the Continuum Hypothesis decided, and algebraic discrete physics — all formally verified.

ID4 ID5

Module 2: The Integer Math Library

EGPTMath — Integer-Only Mathematics — The practical answer to scalable AI. An integer-only math library that replaces floating-point operations (FLOPs) with integer operations (IOPs) — exact arithmetic with no error accumulation. Includes the PPF number representation, vector algebra, polynomial operations, and the pedagogical Faster Abadir Transform (integer-only FFT/QFT).
