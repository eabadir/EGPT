## Methodology

### Hardness Threshold

All problems must exceed **N³ iterations** for ALL three solvers:

| Solver | Description | Iteration Counting |
|--------|-------------|-------------------|
| **Brute Force** | Exhaustive search through all 2^N assignments | Each assignment tested = 1 iteration |
| **Random Search** | Random assignment sampling (50 runs, averaged) | Each random assignment = 1 iteration |
| **DPLL** | Davis-Putnam-Logemann-Loveland with unit propagation | Each recursive call = 1 iteration |

### Verification Process

1. **Problem Generation**: Create candidate problems using known-hard SAT patterns:
   - Hard 3-SAT at phase transition (clause ratio ≈ 4.26)
   - Graph coloring with high chromatic number
   - Forced ordering patterns (DPLL-resistant)
   - Mixed patterns combining multiple techniques

2. **Brute Force Testing**: Run exhaustive search (may timeout for large N)

3. **Random Search Testing**: Run 50 independent trials and compute statistics

4. **DPLL Testing**: Run with unit propagation and pure literal elimination

5. **Validation**: Problem passes only if ALL solvers exceed N³ bound

## Appendix: Methodology Details

### Why N³ Threshold?

The N³ threshold was chosen because:

1. **Clearly exponential**: For N=14, N³=2,744 vs 2^N=16,384 (6x difference)
2. **Testable range**: Allows verification within reasonable time limits
3. **Conservative**: Ensures problems are genuinely hard, not borderline
4. **Educational**: Clear demonstration of polynomial (N²) vs cubic (N³) vs exponential (2^N)

### Pattern Sources

The problem generation patterns are based on:

- **Phase transition 3-SAT**: Mitchell, Selman, & Levesque (1992)
- **Graph coloring**: Known NP-Complete problem encoding
- **Forced ordering**: Resistance to DPLL heuristics (Pipatsrisawat & Darwiche, 2007)
- **Mixed patterns**: Combining multiple hard structures

### Validation

Each problem was verified by:
1. Running brute force solver (up to 2^N assignments)
2. Running random search 50 times and averaging
3. Running DPLL with unit propagation
4. Checking all solvers exceeded N³ threshold
5. For SAT problems, verifying provided certificate
